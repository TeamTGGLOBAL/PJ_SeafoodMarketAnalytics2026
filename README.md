# PJ_SeafoodMarketAnalytics2026
株式会社TG GLOBALの事業計画を作り直すための調査・戦略設計リポジトリです。  
輸出業界の動向、豊洲市場の競争構造、そしてTG GLOBALの再成長シナリオを定義します。

## 目的
- 業界全体のトレンドと構造変化を可視化する
- 豊洲市場の輸出戦略と主要事業者を把握し、競争上の現実を前提化する
- TG GLOBALが勝てるポジションを見極め、投資を受けられる成長戦略に落とし込む

## 背景（危機認識）
豊洲市場は、閉鎖型施設・高度な衛生管理・近隣空港アクセスを背景に、  
「国内の仲卸」から「グローバル輸出ハブ」へ急速に進化しています。  
この動向は、資金力と知名度で劣後するTG GLOBALにとって、  
既存の戦い方が通用しないことを意味します。

## 調査結果の要約（豊洲市場の輸出構造）
- **統括組織の形成**: TEPC（豊洲輸出連携協議会）が「オール豊洲」の輸出連携を主導
- **DXの導入**: クロスマートが受発注をデジタル化し、輸出オペレーションの標準化が進行
- **強者仲卸の多層化**: 旭水産・やま幸・鈴富・米川水産・山八などが役割分担で輸出を牽引
- **青果の拡張**: 水産に加え、青果輸出もエコシステム化し、豊洲が全国産地のゲートウェイ化
- **物流インフラの完成度**: 豊洲物流・輸出利便化施設・空港近接でコールドチェーンが前提化

## TG GLOBALの戦略的ポジション（仮説）
豊洲の「スケール優位」に正面対抗するのではなく、  
**北海道・産地起点の価値設計**と**小回りの効く輸出設計**で差別化する。
- **産地ストーリー型の高付加価値輸出**（希少性・季節性・職人加工）
- **小口混載の最適化**（小規模バイヤーに強い供給形態）
- **データドリブンな相場・需要分析**（既存のGAS/AI基盤の強化）
- **海外BtoBに強い営業体制**（既存販路の深掘り＋市場開拓）

## 直近の焦点（プレゼンと検証）
- 堀江貴文向けの事業プレゼン（10枚構成）の完成
- ターゲットはマレーシア／クアラルンプルに集中
- 初期の取り扱い商品は活ホタテ／冷凍ホタテ／活蟹
- 北海道から需要都市へ直送し、データで改善サイクルを回す
- 現地はBPO体制と通関会社の協力を活用し、冷蔵冷凍・配送はパートナー連携

## 次の打ち手（本リポジトリで詰める項目）

- 輸出競合のマップ化（カテゴリ別：仲卸、物流、DX、商社）
- 成長ストーリーとKPI（12〜24ヶ月）の設計
